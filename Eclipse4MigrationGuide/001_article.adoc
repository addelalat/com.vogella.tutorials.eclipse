Migrating Eclipse 3.x components and applications to Eclipse 4.
This tutorial gives an overview on how to migrate Eclipse
components and application based on the Eclipse 3.x application
model to Eclipse 4. This also includes using the
compatibility
layer
in the
Eclipse platform to run plug-ins
based on the Eclipse
3.x API.
