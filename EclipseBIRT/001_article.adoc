Eclipse Birt.
This tutorial describes how to use Eclipse BIRT for reporting on simple Java Objects (POJO's). 
The tutorial explains also how to deploy the resulting BIRT report into a
webcontainer (Tomcat) and how to use it in an Eclipse RCP application. Eclipse 3.7 (Indigo) is used for this tutorial.
