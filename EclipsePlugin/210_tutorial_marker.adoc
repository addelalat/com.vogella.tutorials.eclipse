== Tutorial: Create Markers for resources

Create a plug-in project "de.vogella.plugin.markers".
Add the dependency to org.eclipse.core.resources", "org.eclipse.jdt.core" and "org.eclipse.jdt.ui".
Create the command "de.vogella.plugin.markers.AddMarker" with the default handler `AddMarkerHandler` in the `de.vogella.plugin.markers.handler` class and add this command to the menu.
