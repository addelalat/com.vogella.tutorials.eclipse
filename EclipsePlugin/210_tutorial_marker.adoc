== Tutorial: Create Markers for resources

Create a plug-in project _de.vogella.plugin.markers_.
Add the dependency to org.eclipse.core.resources", _org.eclipse.jdt.core_ and _org.eclipse.jdt.ui_.
Create the command _de.vogella.plugin.markers.AddMarker_ with a handler `AddMarkerHandler` in the `de.vogella.plugin.markers.handler` class and add this command to the menu.
