Eclipse RAP Tutorial.
This tutorial describes how to create Eclipse RAP applications and how to use the CSS theming to modify their look.
Eclipse RAP aims to enable developers to build rich, Ajax-enabled web applications by using the Eclipse development model.
This tutorial is based on Eclipse 3.7 (Indigo).
