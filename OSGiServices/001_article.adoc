OSGi services with Eclipse Equinox.
This tutorial explains the usage of OSGi services primary focusing on the usage of declarative services.
Eclipse Equinox is used as an stand-alone OSGi server. 
For this tutorial Eclipse 4.4 (Luna) is used.
