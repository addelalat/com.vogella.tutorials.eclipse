== Using SWT in a plug-in project
	
The SWT library
is packaged as an Eclipse
plug-in.
If you create an
Eclipse plug-in and want to use SWT you have to specify a dependency
to the
`org.eclipse.swt`
plug-in
in the corresponding manifest file.

----
